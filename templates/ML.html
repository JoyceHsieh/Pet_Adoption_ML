Supervised Model -- Random Forest

Seclect the input feature: predict the adoption day!!


For this project, we decided to utilize a classifier model instead of a regression model, specifically the Random Forest Classifier after multiple testings. This classifier works by creating multiple uncorrelated individual trees and essentially operates as one ensemble. Although it seems unproductive, having a large sample of uncorrelated trees provides a higher rate of success because it gives the model more chances to be correct. With the amount of different variables in our data, we believed this model would be perfect for our data.

Unsupervised Model -- K-MEAN

To predict and visualize the data!!


We decided to use PCA, Principal Component Analysis, as our model for this section. This model essentially finds patterns within data and dimensionally reduces it. This was perfect for our data because we had a large amount of columns that were not high in importance. By allowing PCA to score each column with a variance value, we can reduce the amount of columns needed to have all of the variance explained!
